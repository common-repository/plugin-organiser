A lightweight plugin filter and organiser. This plugin uses javascript for filtering and
will not affect or modify your installations plugin data.

== Description ==

Plugin Organiser is a simple plugin filter that uses javascript to filter your plugins into groups. Any underlying plugin data is untouched and safe. Create and assign any plugin into as many groups as you want.

== Frequently Asked Questions ==

= I'm getting unexpected results =
As this plugin is written in javascript, you cant have pagination enabled for your plugin page. In the Screen Options of the plugin page, set it to the default of 999.
